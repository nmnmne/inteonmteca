# inteonmteca

Minimal mysterious landing page for the `inteonmteca` music label.

## Structure

- `index.html` - single-page layout
- `styles.css` - visual system, atmosphere, responsive styling
- `script.js` - small ambient interactions
- `assets/logo-inteonmteca.svg` - refined vector logo mark

## Run

Open `index.html` directly in the browser for a quick preview.
